## 🚀 Uruchomienie

```bash
git clone <https://github.com/matefio17/Python_Projects>
cd Python_Projects/Python_CLI_Casino_Simulator
python_cli_casino_simulator.py
```

- Używanie modułu (`random`) do gier losowych
